# Images des projets

Pour que les images s'affichent, ajoute des captures d'écran de chaque projet ici.

## Format recommandé
- **Dimensions** : 800x600px (ratio 4:3)
- **Format** : JPG ou WebP
- **Qualité** : Optimisé pour le web (~100-200kb)

## Comment créer les captures

### Pour les sites en ligne :
1. Va sur chaque site live
2. Fais une capture d'écran de la page d'accueil
3. Redimensionne à 800x600px
4. Compresse avec [squoosh.app](https://squoosh.app) ou [tinypng.com](https://tinypng.com)
5. Renomme selon le tableau ci-dessus

### Pour les sites non disponibles :
Tu peux utiliser :
- Des captures d'écran archivées (si tu les as)
- Des mockups représentatifs du projet
- Ou laisser les icônes placeholder par défaut

## Alternative : Images placeholder

En attendant les vraies images, le site affiche des icônes par défaut selon la catégorie du projet.
Les projets sans lien affichent "Site non disponible" au survol.
